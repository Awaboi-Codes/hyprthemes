## Installation
**Manual Installation**

Required Dependencies: bash, git, make

To download those dependencies:
```bash
sudo pacman -S bash git make
```

Install using git:
```bash
git clone https://github.com/Awaboi-Codes/hyprthemes.git
cd hyprthemes
make build.make
```

**AUR (Arch User Repository)**

*Only Arch Linux users should use this method*

Install using yay (make sure it is hyprthemes, not hyprtheme):
```bash
yay -S hyprthemes
```
